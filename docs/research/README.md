# Research notes

RFCs (0009 federation, 0010 libMLIR FFI + memory safety, 0011 async)
are tracked in our private internal repo, not here.

The public surface for design decisions is:
- `docs/rfcs/*.md` — normative RFC drafts (load-bearing decisions)
- `CHANGELOG.md` — what shipped and when
- `mindlang.dev` — the roadmap card status

This directory is intentionally empty for public review.
